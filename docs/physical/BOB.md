# BOB — The Brass Plumb Bob

```
A machined brass weight on a string.
It does not compute. It does not connect.
It hangs. That is enough.
```

---

## The artifact

A plumb bob, machined from 360 brass. 150 grams. Weighted to hang within 0.1mm of true vertical. Engraved with the Plumb wordmark on one face, "quiet pipes for noisy agents" on the opposite.

```
  _
 / \
|   |  → "PLUMB"
|   |
|   |  → "quiet pipes for noisy agents"
|   |
|   |
 ●
```

It is not for sale. It is not available at conferences. It is earned.

---

## How to earn one

Report a real bug that lands in the conformance suite. The bug must be:
- Reproducible with a fixture
- Fixed with a test that fails before the fix and passes after
- Not a typo, not a cosmetic issue, not a feature request

The first 100 reporters receive a bob. Mailed anywhere. Free. No shipping charge.

---

## What it means

The plumb bob represents the thing Plumb cannot be bought to do: tell the truth about vertical.

- You cannot persuade a plumb bob to hang at an angle.
- You cannot accelerate a plumb bob's settling speed.
- You cannot add features to a plumb bob.

A plumb bob is complete. It does one thing. It does it within the tolerances of gravity.

Plumb is the same. The adapter contract is the string. The ledger is the weight. Gravity is the protocol.

---

## On a desk

A brass plumb bob on a desk means:
- You found a bug in something that was designed not to have bugs
- You contributed a fixture that will catch the same bug in perpetuity
- You are part of the group that made Plumb more reliable

It is not a trophy. It is a certificate of participation in making the protocol gap smaller. When Plumb dissolves into a standards body, the bobs become the only remaining artifacts. They will be worth nothing and everything.

---

## The first bob

The first brass plumb bob is machined from the same brass stock and engraved with the same settings as all future bobs. It is not special. It is not "number 1." It is the same quality as every other bob.

Gravity does not distinguish between first and hundredth. Neither does Plumb.
